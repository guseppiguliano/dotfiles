### Lists

- [ ] You can create checkboxes.
- [x] They can also be checked.
  - [x] Nested lists can also have checkboxes.

### Code

You format small code snippets with `backticks` or `<code>` tags.

Characters that can be escaped with backslash:
```
\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
```
